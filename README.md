# Personal Finance Tracker (C# Console App)

A C# console application for tracking income and expenses with categorized transactions and CSV persistence.

## Features
- Add income and expense transactions
- Categorized income and expense types
- View all transactions
- View financial summaries (totals and balance)
- Save and load transactions using CSV files
- Input validation and exception handling

## Technologies
- C#
- .NET Console Application
- LINQ
- File I/O (CSV)

## How to Run
1. Clone the repository
2. Open `PersonalFinanceTracker.sln` in Visual Studio
3. Run the application
4. Use the console menu to interact with the program

## Notes
- CSV parsing is simplified (commas inside descriptions are not escaped)
- Designed as a learning and portfolio project
